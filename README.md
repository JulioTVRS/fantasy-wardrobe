# 🎭 Fantasy Wardrobe

Sistema de gestão para uma locadora de roupas e fantasias desenvolvido em Python.

## 📖 Sobre

O **Fantasy Wardrobe** é um projeto acadêmico desenvolvido para a disciplina **Algoritmos e Lógica de Programação** da **Universidade Federal do Rio Grande do Norte (UFRN)**.

O objetivo do projeto é aplicar conceitos de programação, lógica computacional e desenvolvimento de sistemas por meio da criação de uma aplicação para gerenciamento de uma locadora de roupas e fantasias.
